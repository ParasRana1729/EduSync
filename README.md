# EduSync

A full-stack educational platform built with React and Express.

## Tech Stack

**Client:**
- React 19
- Vite
- Tailwind CSS
- Radix UI
- React Router DOM

**Server:**
- Express
- JWT Authentication
- bcryptjs

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository
2. Install server dependencies:
   ```bash
   cd server
   npm install
   ```
3. Install client dependencies:
   ```bash
   cd client
   npm install
   ```

### Running the App

Start the server:
```bash
cd server
npm start
```

Start the client:
```bash
cd client
npm run dev
```

The client runs on `http://localhost:5173` by default.
